## About

Most portfolios are a list of things someone claims to know. This one's a working example of what happens when you actually sit down and build — pixel by pixel, line by line, no shortcuts, no copy-paste templates. Every animation, every spacing decision, every color choice was a deliberate call, not a default.

That's the mindset behind everything I do — whether it's wrangling a messy dataset into something a business can act on, or writing CSS for a hover state nobody will consciously notice but everyone will feel. Good work is invisible when it's done right. This site is proof I care about that kind of detail.

---

## Design Choices

- **Palette:** deep navy `#0A0F1E` background, warm amber `#C8922A` accent — confident, not flashy
- **Typography:** `Playfair Display` for headings, `Inter` for body — editorial feel, not a template
- **Motion:** scroll-triggered fade-ins via `IntersectionObserver`, nothing distracting
- **Nav:** sticky blur-backed navbar, collapses to hamburger on mobile
- **Icons:** real SVG brand marks for GitHub and LinkedIn — no emoji anywhere
